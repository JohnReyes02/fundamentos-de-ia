# Fundamentos de la IA

Este repositorio explora los conceptos fundamentales de la Inteligencia Artificial, incluyendo el uso de grandes modelos de lenguaje (LLMs) y la ética en la IA.

## Descripción
Descubre los fundamentos de la IA, aprende a aprovechar herramientas como ChatGPT de forma eficaz en el trabajo y comprende los principios detrás de la IA Generativa.

## Módulos

1.  **Introduction to AI for Work**: Introducción a la IA en el entorno laboral.
2.  **Comprender ChatGPT**: Uso y funcionamiento.
3.  **Comprender el Machine Learning**: Conceptos base.
4.  **Conceptos de Grandes Modelos Lingüísticos (LLM)**: Funcionamiento de modelos de texto.
5.  **Conceptos de la IA Generativa**: Creación de contenido con IA.
6.  **Ética de la IA**: Consideraciones éticas y responsables.

## Tecnologías y Formato

*   **Quarto**: Utilizado para estructurar el libro/sitio web.
*   **Enfoque**: Teórico y práctico sobre conceptos modernos de IA.

## Configuración del Entorno

1.  **Crear el entorno virtual**:
    ```bash
    python3 -m venv .venv
    ```
2.  **Activar el entorno**:
    *   macOS/Linux: `source .venv/bin/activate`
    *   Windows: `.venv\Scripts\activate`
3.  **Instalar dependencias**:
    ```bash
    pip install -r requirements.txt
    ```
